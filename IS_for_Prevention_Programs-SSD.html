Information System for Prevention Programs

CHSR is working with the New York State Office of Children and Family Services (OCFS) to design and develop an information system for a number of OCFS-supported programs.

Systems & Software Development

CHSR is working with OCFS to design and develop an information system for the following OCFS-supported programs: Kinship Care, Prevention, Post Adoption, CPS/DV, and Trust Fund initiatives. The OCFS contractors supported through these prevention programs will have access to a newly-developed Internet-based system to input quarterly data. The system will produce reports that can be run by county, by program, and by time period. Initially, the system will be designed to report on the number and characteristics of participants enrolled and served. Later versions will incorporate reporting by performance targets.
